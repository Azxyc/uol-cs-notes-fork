## Proving Existential Statements
An existential statement is a statement in the form: 
{% raw %}<pre>\[\exists x\ Q(x)\]</pre>{% endraw %}
This means that there exists a value to which the function {% raw %}<pre>\(Q(x)\)</pre>{% endraw %} holds true. This may be under additional parameters.

The easiest way to prove this is to find an {% raw %}<pre>\(x\)</pre>{% endraw %} that makes the function {% raw %}<pre>\(Q(x)\)</pre>{% endraw %} true. Not all can be proved this way. 

### Examples
1. {% raw %}<pre>\(\exists\)</pre>{% endraw %} an even integer {% raw %}<pre>\(n\)</pre>{% endraw %} that can be written in two ways as a sum of two prime numbers. 
	{% raw %}<pre>\[10=5+5=7+3\]</pre>{% endraw %}
1. There {% raw %}<pre>\(\exists\)</pre>{% endraw %} integers {% raw %}<pre>\(m\)</pre>{% endraw %} and {% raw %}<pre>\(n\)</pre>{% endraw %} such that {% raw %}<pre>\(m>1\)</pre>{% endraw %}, {% raw %}<pre>\(n>1\)</pre>{% endraw %} and {% raw %}<pre>\(\frac{1}{m} + \frac{1}{n}\)</pre>{% endraw %} is an integer.
	{% raw %}<pre>\[m=n=2\]</pre>{% endraw %}
		Giving an example is a suitable proof.

## Proving Universal Statements
Generally proofs will require you to answer a universal statement rather than an existential one. an existential statement is of the form:
{% raw %}<pre>\[\forall x \text{ if } P(x) \text{ then } Q(x)\]</pre>{% endraw %}
This means that for all of {% raw %}<pre>\(x\)</pre>{% endraw %} if one function is applied to {% raw %}<pre>\(x\)</pre>{% endraw %} another function in {% raw %}<pre>\(x\)</pre>{% endraw %} also holds true. For example:

If {% raw %}<pre>\(a\)</pre>{% endraw %} and {% raw %}<pre>\(b\)</pre>{% endraw %} are integers then {% raw %}<pre>\(6a^2b\)</pre>{% endraw %} is even.
{:.info}

In this statement the "{% raw %}<pre>\(a\)</pre>{% endraw %} and {% raw %}<pre>\(b\)</pre>{% endraw %} are integers" count for {% raw %}<pre>\(P(x)\)</pre>{% endraw %} and "{% raw %}<pre>\(6a^2b\)</pre>{% endraw %} is even" counts for {% raw %}<pre>\(Q(x)\)</pre>{% endraw %}. 

{% raw %}<pre>\(6a^2b\)</pre>{% endraw %}  

{% raw %}<pre>\(2(3a^2b)\)</pre>{% endraw %}

By halving you are proving that the answer is even as it is a multiple of two.

### Proof by Exhaustion
For theorems examining a relatively small number of examples you can test each value to see if the statement holds true. That is proof by exhaustion.

#### Example
* Prove that {% raw %}<pre>\((n+1)^3 \geq 3^n\)</pre>{% endraw %} if {% raw %}<pre>\(n\)</pre>{% endraw %} is a positive number.

As this theorem has such a small scope then each value can be tested to see if it is correct.

### Generalising from the Generic Particular
This method allows for using algebra and known rules to prove a statement generally.

#### Method
* Express the statement to be provided in the form {% raw %}<pre>\(\forall x,\text{ if } P(x) \text{ then } Q(x)\)</pre>{% endraw %}
* Start the proof by supposing {% raw %}<pre>\(x\)</pre>{% endraw %} is a particular by arbitrarily chosen element for which the hypothesis {% raw %}<pre>\(P(x)\)</pre>{% endraw %} is true.
* Show that the conclusion {% raw %}<pre>\(Q(x)\)</pre>{% endraw %} is true by using definitions, previously established results, and the rules for logical inference.

```mermaid
graph LR
Definitions --> A[Conjecture]
B[Known Facts] --> A
```

This method brings together definitions and facts into a conjecture.
{:.info}

### Example
Prove that the sum of any two even integers is even.

1. (Assume that/Suppose that) {% raw %}<pre>\(m\)</pre>{% endraw %} and {% raw %}<pre>\(n\)</pre>{% endraw %} are particular but arbitrarily chosen even integers.
1. As we assumed that {% raw %}<pre>\(m\)</pre>{% endraw %} is an even integer, {% raw %}<pre>\(m = 2k\)</pre>{% endraw %} for some integer {% raw %}<pre>\(k\)</pre>{% endraw %}. 
1. Likewise {% raw %}<pre>\(n\)</pre>{% endraw %} is an even integer, {% raw %}<pre>\(n = 2l\)</pre>{% endraw %}, for some integer {% raw %}<pre>\(l\)</pre>{% endraw %}
	* We cannot use the same letter again as {% raw %}<pre>\(k\)</pre>{% endraw %} has already been used
1. Then {% raw %}<pre>\(m+n=2k+2l=2(k+l)\)</pre>{% endraw %}, which is even as {% raw %}<pre>\(k+l\)</pre>{% endraw %} is an integer.

The final step is called the conjecture as is aided by the previous steps to explain why the conjecture holds true.

### Disproving Universal Statement by Counterexample
To disprove a statement means to show that it is false. For example, for a statement such as:
{% raw %}<pre>\[\forall x \text{ if } P(x) \text{ then } Q(x)\]</pre>{% endraw %}
You are saying that the opposite is true:
{% raw %}<pre>\[\exists x \text{ such that } P(x) \text{ and not } Q(x)\]</pre>{% endraw %}

This means that you must give at least one example that disproves the universal statement.

#### Example
Is it true that for every positive integer {% raw %}<pre>\(n,n^2\geq 2n\)</pre>{% endraw %}?

No as for {% raw %}<pre>\(n=1,\ n^2 = 1\)</pre>{% endraw %} and {% raw %}<pre>\(2n=2\)</pre>{% endraw %} which is greater than {% raw %}<pre>\(n^2\)</pre>{% endraw %}
		
You can correct this by stipulating that for ever integer greater than one the statement holds true.
{:.info}
