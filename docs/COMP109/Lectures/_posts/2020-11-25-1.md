## Digraph Representation
In the directed graph representation, {% raw %}<pre>\(R\)</pre>{% endraw %} is:

* **Reflexive** if there is always an arrow from every vertex to itself.
* **Symmetric** if whenever there is an arrow from {% raw %}<pre>\(x\)</pre>{% endraw %} to {% raw %}<pre>\(y\)</pre>{% endraw %} there is also an arrow from {% raw %}<pre>\(y\)</pre>{% endraw %} to {% raw %}<pre>\(x\)</pre>{% endraw %}.
* **Antisymmetric** if  whenever there is an arrow from {% raw %}<pre>\(x\)</pre>{% endraw %} to {% raw %}<pre>\(y\)</pre>{% endraw %} and {% raw %}<pre>\(x\neq y\)</pre>{% endraw %}, then there is no arrow from {% raw %}<pre>\(y\)</pre>{% endraw %} to {% raw %}<pre>\(x\)</pre>{% endraw %}.
* **Transitive** if whenever there is an arrow from {% raw %}<pre>\(x\)</pre>{% endraw %} to {% raw %}<pre>\(y\)</pre>{% endraw %} and from {% raw %}<pre>\(y\)</pre>{% endraw %} to {% raw %}<pre>\(z\)</pre>{% endraw %} there is also an arrow from {% raw %}<pre>\(x\)</pre>{% endraw %} to {% raw %}<pre>\(z\)</pre>{% endraw %}.

### Example 1
Let {% raw %}<pre>\(A=\{1,2,3\},R_1=\{(1,1),(2,2),(3,3),(2,3),(3,2)\}\)</pre>{% endraw %}

```mermaid
graph TD
1((1)) --> 1
2((2)) --> 2
3((3)) --> 3
2 --> 3
3 --> 2
```

* Reflective {% raw %}<pre>\(\forall x:xRx\)</pre>{% endraw %}
	* True
* Symmetric {% raw %}<pre>\(\forall x,y: xRy\Rightarrow yRx\)</pre>{% endraw %}
	* True
		
	If two items, such as 1 and 2, are not connected they are not obligated to connect back. The lack of a connection doesn't break this property.
	{:.info}
* Antisymmetric {% raw %}<pre>\(\forall x,y:xRy,yRx\Rightarrow x=y\)</pre>{% endraw %}
	* False
* Transitive {% raw %}<pre>\(\forall x,y,z:xRy,yRz\Rightarrow xRz\)</pre>{% endraw %}
	* True

### Example 2
Let {% raw %}<pre>\(A=\{1,2,3\},R_1=\{(2,2),(2,3),(3,2),(3,3)\}\)</pre>{% endraw %}

```mermaid
graph TD
1((1))
2((2)) --> 2
3((3)) --> 3
2 --> 3
3 --> 2
```

* Reflective {% raw %}<pre>\(\forall x:xRx\)</pre>{% endraw %}
	* False
* Symmetric {% raw %}<pre>\(\forall x,y: xRy\Rightarrow yRx\)</pre>{% endraw %}
	* True
* Antisymmetric {% raw %}<pre>\(\forall x,y:xRy,yRx\Rightarrow x=y\)</pre>{% endraw %}
	* False
	
	If there are two nodes with a double arrow then this property is automatically broken.
	{:.info}
* Transitive {% raw %}<pre>\(\forall x,y,z:xRy,yRz\Rightarrow xRz\)</pre>{% endraw %}
	* True

### Example - Reachability
Consider some roads in a city. Some may be private and not link up, some may be one way, some may have no stopping. This is represented on the following graph:

```mermaid
graph TD
1((1)) --> 2((2))
2 --> 6((6))
6 --> 4((4))
4 --> 5((5))
5 --> 5
4 --> 1
1 --> 4
3((3)) --> 3
```

Say you start at `1` and end at `5` are you able to make this path?

If there was a transitive relation from the source to the destination then you would be able to get there in one hop.
