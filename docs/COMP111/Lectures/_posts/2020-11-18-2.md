An event is a subset {% raw %}<pre>\(E ⊆ S\)</pre>{% endraw %} of the sample space {% raw %}<pre>\(S\)</pre>{% endraw %}. The probability of the even {% raw %}<pre>\(E\)</pre>{% endraw %} is given by:

{% raw %}<pre>\[P(E)=\sum_{x\in E}P(x)\]</pre>{% endraw %}

* {% raw %}<pre>\(0 ≤ P(E) ≤ 1\)</pre>{% endraw %} for every event {% raw %}<pre>\(E\)</pre>{% endraw %}
* {% raw %}<pre>\(P(\emptyset) = 0\)</pre>{% endraw %} and {% raw %}<pre>\(P(S) = 1\)</pre>{% endraw %}

## Example - Fair Dice
If I roll a die three times, the event {% raw %}<pre>\(E\)</pre>{% endraw %} of rolling at least one 6 is given by:

* The set of sequences of length 4 over {% raw %}<pre>\(\{1,\ldots,6\}\)</pre>{% endraw %} containing at least one 6.
* {% raw %}<pre>\(P(E)\)</pre>{% endraw %} is the number of sequences containing at least one 6 divided by {% raw %}<pre>\(6\times6\times6\times6=216\)</pre>{% endraw %}.

If we roll a fair die, then the event E of rolling an odd number is given by:

* The set {% raw %}<pre>\(E=\{1,3,5\}\)</pre>{% endraw %}
* {% raw %}<pre>\(P(E)=P(1)+P(3)+P(5)=\frac{1}{6}+\frac{1}{6}+\frac{1}{6}=\frac{1}{2}\)</pre>{% endraw %}

## Probability of Composed Events
* The complement of an event can be computed from the probability of the event.
* The union of events can be computed from the probabilities of the individual events.

### Complement
Let {% raw %}<pre>\(\neg E = S - E\)</pre>{% endraw %}. Then {% raw %}<pre>\(P(\neg E)=1-P(E)\)</pre>{% endraw %}

![Venn Diagram 1]({{site.baseurl}}/assets/COMP111/Lectures/2020-11-18-2-1.png)

Additionally, {% raw %}<pre>\(S=\neg E\cup E\)</pre>{% endraw %}.

#### Proof

{% raw %}<pre>\[1=\sum_{x\in S}P(x)=\sum_{x\in E}P(x)+\sum_{x\in \neg E}P(x)\]</pre>{% endraw %}

Thus,

{% raw %}<pre>\[\sum_{x\in\neg E}P(x)=1-\sum_{x\in E}P(x)\]</pre>{% endraw %}

#### Example
What is the probability that at least one bit in a randomly generated sequence of 10 bits is 0?

* {% raw %}<pre>\(S = \{0, 1\}^{10} =\)</pre>{% endraw %} all sequences of 0 and 1 of length 10.
* For every {% raw %}<pre>\(x ∈ S, P(x) = (\frac{1}{2})^{10} = \frac{1}{2^{10}}\)</pre>{% endraw %}.
* {% raw %}<pre>\(E =\)</pre>{% endraw %} all sequences of 0 and 1 of length 10 containing at least one 0.
* {% raw %}<pre>\(\neg E=\{1111111111\}\)</pre>{% endraw %}
* {% raw %}<pre>\(P(\neg E)=\frac{1}{2^{10}}\)</pre>{% endraw %}
* {% raw %}<pre>\(P(E)=1-\frac{1}{2^{10}}\)</pre>{% endraw %}

### Union
{% raw %}<pre>\[P(E_1\cup U_2)=P(E_1)+P(E_2)-P(E_1\cap E_2)\]</pre>{% endraw %}

![Venn Diagram 2]({{site.baseurl}}/assets/COMP111/Lectures/2020-11-18-2-2.png)

Additionally, {% raw %}<pre>\(\vert E_1\cup E_2\vert  = \vert E_1\vert +\vert E_2\vert -\vert E_1\cap E_2\vert \)</pre>{% endraw %}

#### Proof

* {% raw %}<pre>\(P(E_1)=\sum_{x\in E_1}P(x)\)</pre>{% endraw %}
* {% raw %}<pre>\(P(E_2)=\sum_{x\in E_2}P(x)\)</pre>{% endraw %}
* {% raw %}<pre>\(P(E_1\cup E_2)=\sum_{x\in E_1\cup E_2}P(x)\)</pre>{% endraw %}

Thus,

{% raw %}<pre>\[
\begin{aligned}
P(E_1\cup E_2)&=\sum_{x\in E_1\cup E_2}P(x)\\
&=\sum_{x\in E_1}P(x)+\sum_{x\in E_3}P(x)-\sum_{x\in E_1\cup E_2}P(x)\\
&=P(E_1)+P(E_2)-P(E_1\cap E_2)
\end{aligned}
\]</pre>{% endraw %}

#### Example
Suppose I have a jar of 30 sweets:

\vert  \vert  Red \vert  Blue \vert  Green \vert 
\vert  :-: \vert  :-: \vert  :-: \vert  :-: \vert 
\vert  Circular \vert  2 \vert  4 \vert  3 \vert 
\vert  Square \vert  6 \vert  7 \vert  8 \vert 

The sample space {% raw %}<pre>\(S\)</pre>{% endraw %} has 30 elements and if one chooses a sweet uniformly at random then then the probability for all {% raw %}<pre>\(x\in S\)</pre>{% endraw %} is:

{% raw %}<pre>\[P(x)=\frac{1}{30}\]</pre>{% endraw %}

What is the probability of choosing a red or circular sweet?

* The probability that it is red is {% raw %}<pre>\(\frac{2+6}{30}=\frac{8}{30}(P(R)=\frac{8}{30})\)</pre>{% endraw %}
* The probability that it is cicular is {% raw %}<pre>\(\frac{2+4+3}{30}=\frac{9}{30}(P(C)=\frac{9}{30})\)</pre>{% endraw %}

Then {% raw %}<pre>\(P(R\cup C)\)</pre>{% endraw %} is the probability that the sweet is red or circular:

{% raw %}<pre>\[P(R\cup C) = P(R)+P(C)-P(R\cap C) = \frac{8}{30}+\frac{9}{30}-\frac{2}{30}=\frac{15}{30}=\frac{1}{2}\]</pre>{% endraw %}
